Google Sign in LibGDX

2018-08-03
LibGDX Games

Contents

    • Introduction
    • Pre-requisites
    • Installation

Introduction

This tutorial will show you how to integrated google sign in in LibGDX. I assume that you are familiar with LibGDX, cross platform gaming framework. I’ll explain how integrate Google Sign in iOS.

This is based on MobiDevelop’s Google Sign-in Robopods.

Pre-requisites

It is assumed that you have:

  • Created a LibGDX application using the standard gdx-setup wizard
  • Get an OAuth client ID & URL scheme for your project from here
  • Download 3rd party native framework for your iOS module, instruction here

Installation

  1. Add downloaded framework to your iOS project’s libs/ folder (create If not present there)

  2. Put the GoogleSignIn.bundle in your iOS project’s resources/ folder (create If not present there)

  3. Add the following to your robovm.xml

    <config>
...
<resources>
    ...
    <resource>
        <directory>resources</directory>
    </resource>
</resources>
...
<frameworkPaths>
    <path>libs</path>
</frameworkPaths>
<frameworks>
    <framework>GoogleSignIn</framework>
</frameworks>
</config>

  4. Add the following to your info.plist.xml

    <key>CFBundleURLTypes</key>
<array>
  <dict>
  <key>CFBundleTypeRole</key>
        <string>Editor</string>
        <key>CFBundleURLName</key>
        <string>com.googleusercontent.apps.1234567890-qweasdzxcqweasdzxc321456</string>
        <key>CFBundleURLSchemes</key>
        <array>
           <string>com.googleusercontent.apps.1234567890-qweasdzxcqweasdzxc321456</string>
         </array>
     </dict>
     <dict>
            <key>CFBundleTypeRole</key>
            <string>Editor</string>
            <key>${bundle.id}</key>
            <string>${bundle.id}</string>
            <key>CFBundleURLSchemes</key>
            <array>
                <string>${bundle.id}</string>
            </array>
        </dict>
</array>

  5. Add the following dependency to your build.gradle

    dependencies {
     ... other dependencies ...
     compile "com.mobidevelop.robovm:robopods-google-signin-ios:$robopodsVersion"
}

  6. Coding part
